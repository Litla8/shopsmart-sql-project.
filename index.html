Repeat vs One-Time Buyers

Segment customers into Repeat Buyer or One-Time Buyer categories using CASE WHEN and COUNT on order_id.

Sample Query

-- Task 6: Repeat vs One-Time Buyers
SELECT
  buyer_type,
  COUNT(customer_id) AS customer_count
FROM (
  SELECT
    customer_id,
    CASE
      WHEN COUNT(order_id) >= 2 THEN 'Repeat Buyer'
      WHEN COUNT(order_id) = 1  THEN 'One-Time Buyer'
    END AS buyer_type
  FROM orders
  GROUP BY customer_id
) x
GROUP BY buyer_type;
